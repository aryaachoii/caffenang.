📱 Cetak Bluetooth di iPhone / iPad

Safari di iPhone/iPad belum mendukung Web Bluetooth secara langsung — ini keterbatasan dari Apple, bukan dari aplikasi kasir ini. Ada dua cara agar fitur cetak Bluetooth tetap berfungsi di iPhone/iPad Anda:

Opsi 1 — Pakai browser Bluefy atau WebBLE
Install aplikasi Bluefy – Web BLE Browser atau WebBLE dari App Store, lalu buka alamat web sistem kasir ini di dalam aplikasi tersebut (bukan di Safari). Kedua browser ini sudah membawa dukungan Bluetooth sendiri, jadi tombol "Hubungkan" akan langsung berfungsi.
Opsi 2 — Pasang ekstensi iOSWebBLE di Safari
Install ekstensi iOSWebBLE dari App Store, aktifkan lewat Pengaturan > Safari > Ekstensi, lalu izinkan untuk situs kasir ini. Setelah aktif, Safari biasa pun bisa memakai fitur Bluetooth tanpa perlu ganti browser.

Setelah salah satu opsi di atas aktif, kembali ke halaman ini dan tekan tombol "Hubungkan" seperti biasa.
